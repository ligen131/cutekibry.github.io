LGOJ #P2508. 「HAOI2008」圆上的整点

内存限制：256 MiB 时间限制：1000 ms 标准输入输出
题目类型：传统 评测方式：文本比较
上传者：未知
原题链接 查看源代码 修改历史 版本对比 ::after

题目描述

求一个给定的圆 $(x^2+y^2=r^2)$，在圆周上有多少个点的坐标是整数。

输入格式

只有一行，一个整数 $r$，表示圆的半径。

输出格式

只有一行，一个整数，表示答案。

样例

样例输入

4

样例输出

4

数据范围与提示

对于 $100\%$ 的数据，$n \leq 2 \times 10^9$。

题解

主页面：「专题」圆上的整点

直接分解质因数即可。

时间复杂度：$O(\sqrt n)$。

代码

#include <cstdio>

int main() {
#ifndef ONLINE_JUDGE
    freopen("P2508.in", "r", stdin);
    freopen("P2508.out", "w", stdout);
#endif

    int n, i, cnt, ans = 1;

    scanf("%d", &n);
    for (i = 2; i * i <= n; i++) {
        if (!(n % i)) {
            cnt = 1;
            while (!(n % i)) {
                n /= i;
                cnt += 2;
            }
            if (~i & 2)
                ans *= cnt;
        }
    }
    if (n > 1 and ~n & 2)
        ans *= 3;
    printf("%d\n", 4 * ans);
    return 0;
}
