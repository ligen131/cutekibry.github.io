题目描述

原题来自：2012 年国家集训队互测

给你一个无向带权连通图，每条边是黑色或白色。让你求一棵最小权的恰好有 $\text{need}$ 条白色边的生成树。题目保证有解。

输入格式

第一行 $V,E,\text{need}$ 分别表示点数，边数和需要的白色边数。

接下来 $E$ 行，每行 $s,t,c,\text{col}$ 表示这边的端点（点从 $0$ 开始标号），边权，颜色（$0$ 白色，$1$ 黑色）。

输出格式

一行表示所求生成树的边权和。

样例

样例输入

2 2 1
0 1 1 1
0 1 2 0

样例输出

2

数据范围与提示

对于所有数据，$V\le 5\times 10^4,E\le 10^5$，边权为 $[1, 100]$ 中的正整数。

题解

考虑二分把每条白边减去的权值 $cost$，然后做一次 Kruskal 求出忽略颜色的最小生成树。可见，$cost$ 越大，白边的数量就会越少。通过二分，就可以控制 $cost$ 的值使得白边的数量恰好为 $need$ 条；同时也可以求出此时得到的最小生成树边权之和，即可得到答案。

设 $F(x)$ 表示当白边为 $x$ 条时，最小生成树的权值，则在定义域内 $\Delta F(x)$ 单调递增（或者说，$F(x)-F(x-1) \le F(x+1)-F(x)$），即为一个凸函数，那么所有的函数值均位于凸包上，则可二分斜率以切凸包的方法来找到 $F(need)$。（即著名的凸优化，亦有人称 wqs 二分）。

设二分 $cost$ 时选择的白边数量为 $x_0$，边权之和为 $w$，那么有 $F(x_0) = cost \cdot x_0 + w$。此时可以把它看作是一条斜率为 $cost$，截距为 $w$ 的直线 $y = cost \cdot x + w$。

同时注意到：

  • $F(x)$ 是凸函数
  • 我们能保证二分时求出的 $w$ 最优

所以这条直线和这个凸包只会有一个交点；换句话说，这条直线是这个凸包在 $x = x_0$ 处的切线。

那么可以发现，我们二分 $cost$，实际上是在二分切线的斜率。

由于 $F(x)$ 是个凸函数，所以 $cost$ 越大，所切的点的横坐标也就越大（即选择的白边数量 $x_0$ 最大）。所以通过二分 $cost$，就相当于二分凸包上点的横坐标，即 $need$。

时间复杂度：$O(m \log m + n \cdot \alpha(n) \log c)$。

一个小细节：如果在二分后，发现有两棵生成树 $T_1, T_2$ 的权值之和 $w$ 都相同，但选择的白边数量 $x_0, x_1$ 却不同，要怎么处理？

细节留作读者思考。

代码

#include <algorithm>
#include <cstdio>
#include <cstring>

#ifdef ONLINE_JUDGE
const int MAXM = 100000 + 10;
const int MAXN = 50000 + 10;
#else
const int MAXM = 1000 + 10;
const int MAXN = 1000 + 10;
#endif

int n, m, k, m0;
int fa[MAXN], size[MAXN];

struct Edge {
    int s, t, w, col;
    bool operator<(const Edge b) const { return w + 1000 * col < b.w + 1000 * b.col; }
};
Edge e[MAXM];

int gf(int x) { return (x ^ fa[x]) ? (fa[x] = gf(fa[x])) : x; }
inline void init() {
    int i;
    for (i = 1; i <= n; i++) {
        fa[i] = i;
        size[i] = 1;
    }
}
inline void merge(int x, int y) {
    x = gf(x);
    y = gf(y);
    if (x ^ y) {
        if (size[x] < size[y])
            std::swap(x, y);
        size[x] += size[y];
        fa[y] = x;
    }
}

int f(int x, int &totw) {
    int i, j, k, cnt0 = 0;

    for (i = 1; i <= m0; i++) e[i].w -= x;

    init();
    i = 1;
    j = m0 + 1;
    totw = 0;
    while (i <= m0 or j <= m) {
        if ((i <= m0 and e[i].w <= e[j].w) or j > m) {
            k = i;
            i++;
        } else {
            k = j;
            j++;
        }
        if (gf(e[k].s) ^ gf(e[k].t)) {
            totw += e[k].w;
            if (!e[k].col)
                cnt0++;
            merge(e[k].s, e[k].t);
        }
    }

    for (i = 1; i <= m0; i++) e[i].w += x;
    return cnt0;
}

int main() {
#ifndef ONLINE_JUDGE
    freopen("10069.in", "r", stdin);
    freopen("10069.out", "w", stdout);
#endif

    int i, l, r, mid, ans;

    scanf("%d %d %d", &n, &m, &k);
    for (i = 1; i <= m; i++) {
        scanf("%d %d %d %d", &e[i].s, &e[i].t, &e[i].w, &e[i].col);
        e[i].s++;
        e[i].t++;
        if (!e[i].col)
            m0++;
    }
    std::sort(e + 1, e + 1 + m);

    l = -100;
    r = 100;
    while (l ^ r) {
        mid = (l + r) >> 1;

        if (f(mid, ans) >= k)
            r = mid;
        else
            l = mid + 1;
    }
    f(l, ans);
    printf("%d\n", ans + l * k);
    return 0;
}
