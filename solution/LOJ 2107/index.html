LOJ #2107. 「JLOI2015」城池攻占

内存限制：256 MiB 时间限制：1000 ms 标准输入输出
题目类型：传统 评测方式：文本比较
上传者：未知
原题链接 查看源代码 修改历史 版本对比 ::after

题目描述

小铭铭最近获得了一副新的桌游，游戏中需要用 $m$ 个骑士攻占 $n$ 个城池。

这 $n$ 个城池用 $1$ 到 $n$ 的整数表示。除 $1$ 号城池外，城池 $i$ 会受到另一座城池 $f_i$ 的管辖，其中 $f_i <i$。也就是说，所有城池构成了一棵有根树。这 $m$ 个骑士用 $1$ 到 $m$ 的整数表示，其中第 $i$ 个骑士的初始战斗力为 $s_i$，第一个攻击的城池为 $c_i$。

每个城池有一个防御值 $h_i$，如果一个骑士的战斗力大于等于城池的生命值，那么骑士就可以占领这座城池；否则占领失败，骑士将在这座城池牺牲。占领一个城池以后，骑士的战斗力将发生变化，然后继续攻击管辖这座城池的城池，直到占领 $1$ 号城池，或牺牲为止。

除 $1$ 号城池外，每个城池 $i$ 会给出两个战斗力变化参数 $a_i, v_i$。若 $a_i = 0$，攻占城池 $i$ 以后骑士战斗力会增加 $v_i$；若 $a_i =1$，攻占城池 $i$ 以后，战斗力会乘以 $v_i$。注意每个骑士是单独计算的。也就是说一个骑士攻击一座城池，不管结果如何，均不会影响其他骑士攻击这座城池的结果。

现在的问题是，对于每个城池，输出有多少个骑士在这里牺牲；对于每个骑士，输出他攻占的城池数量。

输入格式

第一行包含两个正整数 $n, m$，表示城池的数量和骑士的数量。

第二行包含 $n$ 个整数，其中第 $i$ 个数为 $h_i$，表示城池 $i$ 的防御值。

第三到第 $n+1$ 行，每行包含三个整数。其中第 $i +1$ 行的三个数为 $f_i, a_i, v_i$，分别表示管辖这座城池的城池编号和两个战斗力变化参数。

第 $n +2$ 到 $n + m +1$ 行，每行包含两个整数。其中第 $n + i$ 行的两个数为 $s_i,c_i$，分别表示初始战斗力和第一个攻击的城池。

输出格式

输出 $n + m$ 行，每行包含一个非负整数。其中前 $n$ 行分别表示在城池 $1$ 到 $n$ 牺牲的骑士数量，后 $m$ 行分别表示骑士 $1$ 到 $m$ 攻占的城池数量。

样例

样例输入

5 5
50 20 10 10 30
1 1 2
2 0 5
2 0 -10
1 0 10
20 2
10 3
40 4
20 4
35 5

样例输出

2
0

0

数据范围与提示

对于 $100 \%$ 的数据，$1 \leq n,m \leq 300000, \ 1 \leq f_i<i, \ 1 \leq c_i \leq n, \ -10^{18} \leq h_i,v_i,s_i \leq 10^{18}$；$a_i$ 等于 $0$ 或者 $1$；当 $a_i =1$ 时，$v_i > 0$；保证任何时候骑士战斗力值的绝对值不超过 $10^{18}$。

题解

维护可并最小根堆即可。

可考虑使用左偏树。

时间复杂度：$O(n \log n)$。

代码

#include <algorithm>
#include <cstdio>

typedef long long int64;

#ifdef ONLINE_JUDGE
const int MAXN = 300000 + 10;
#else
const int MAXN = 1000;
#endif

int n, m;

int pre[MAXN], to[MAXN], head[MAXN], wcnt;
int dep[MAXN];
int64 qdef[MAXN], qx[MAXN];
int qopt[MAXN], qans[MAXN];

int st[MAXN], ans[MAXN];
int rt[MAXN];
int h[MAXN], lson[MAXN], rson[MAXN];
int64 addmark[MAXN], mulmark[MAXN];
int64 val[MAXN];

inline void addedge(int u, int v) {
    wcnt++;
    pre[wcnt] = head[u];
    head[u] = wcnt;
    to[wcnt] = v;
}
inline void addnode(int p, int64 x) {
    addmark[p] += x;
    val[p] += x;
}
inline void mulnode(int p, int64 x) {
    addmark[p] *= x;
    mulmark[p] *= x;
    val[p] *= x;
}
inline void down(int p) {
    if (mulmark[p] ^ 1) {
        if (lson[p])
            mulnode(lson[p], mulmark[p]);
        if (rson[p])
            mulnode(rson[p], mulmark[p]);
        mulmark[p] = 1;
    }
    if (addmark[p]) {
        if (lson[p])
            addnode(lson[p], addmark[p]);
        if (rson[p])
            addnode(rson[p], addmark[p]);
        addmark[p] = 0;
    }
}
inline void adjust(int p) {
    if (h[lson[p]] < h[rson[p]])
        std::swap(lson[p], rson[p]);
    h[p] = h[rson[p]] + 1;
}
void merge(int &p, int l, int r) {
    if (!l or !r)
        p = l | r;
    else {
        if (val[l] > val[r])
            std::swap(l, r);
        p = l;
        down(p);
        merge(rson[p], rson[p], r);
        adjust(p);
    }
}
inline void pop(int &p) {
    down(p);
    merge(p, lson[p], rson[p]);
}

#define v to[i]
void dfs(int u) {
    int i;

    for (i = head[u]; i; i = pre[i]) {
        dep[v] = dep[u] + 1;
        dfs(v);
        merge(rt[u], rt[u], rt[v]);
    }

    while (rt[u] and val[rt[u]] < qdef[u]) {
        qans[u]++;
        ans[rt[u]] = dep[st[rt[u]]] - dep[u] + 1;
        pop(rt[u]);
    }
    if (rt[u]) {
        if (!qopt[u])
            addnode(rt[u], qx[u]);
        else
            mulnode(rt[u], qx[u]);
    }
}
#undef v

int main() {
#ifndef ONLINE_JUDGE
    freopen("2107.in", "r", stdin);
    freopen("2107.out", "w", stdout);
#endif

    int i, x;

    scanf("%d %d", &n, &m);
    for (i = 1; i <= n; i++)
        scanf("%lld", &qdef[i]);
    for (i = 2; i <= n; i++) {
        scanf("%d %d %lld", &x, &qopt[i], &qx[i]);
        addedge(x, i);
    }
    for (i = 1; i <= m; i++) {
        scanf("%lld %d", &val[i], &st[i]);
        mulmark[i] = 1;
        merge(rt[st[i]], rt[st[i]], i);
    }

    dep[1] = 1;
    dfs(1);

    for (i = 1; i <= n; i++)
        printf("%d\n", qans[i]);
    for (i = 1; i <= m; i++)
        printf("%d\n", ans[i] ? (ans[i] - 1) : dep[st[i]]);
    return 0;
}
