LOJ #2182. 「SDOI2015」寻宝游戏

内存限制：256 MiB 时间限制：2000 ms 标准输入输出
题目类型：传统 评测方式：文本比较
上传者：未知
原题链接 查看源代码 修改历史 版本对比 ::after

题目描述

小 B 最近正在玩一个寻宝游戏，这个游戏的地图中有 $N$ 个村庄和 $N-1$ 条道路，并且任何两个村庄之间有且仅有一条路径可达。游戏开始时，玩家可以任意选择一个村庄，瞬间转移到这个村庄，然后可以任意在地图的道路上行走，若走到某个村庄中有宝物，则视为找到该村庄内的宝物，直到找到所有宝物并返回到最初转移到的村庄为止。

小 B 希望评测一下这个游戏的难度，因此他需要知道玩家找到所有宝物需要行走的最短路程。但是这个游戏中宝物经常变化，有时某个村庄中会突然出现宝物，有时某个村庄内的宝物会突然消失，因此小 B 需要不断地更新数据，但是小 B 太懒了，不愿意自己计算，因此他向你求助。为了简化问题，我们认为最开始时所有村庄内均没有宝物。

输入格式

第一行，两个整数 $N$、$M$，其中 $M$ 为宝物的变动次数。

接下来的 $N-1$ 行，每行三个整数 $x$、$y$、$z$，表示村庄 $x$、$y$ 之间有一条长度为 $z$ 的道路。

接下来的 $M$ 行，每行一个整数 $t$，表示一个宝物变动的操作。若该操作前村庄 $t$ 内没有宝物，则操作后村庄内有宝物；若该操作前村庄 $t$ 内有宝物，则操作后村庄内没有宝物。

输出格式

$M$ 行，每行一个整数，其中第 $i$ 行的整数表示第 $i$ 次操作之后玩家找到所有宝物需要行走的最短路程。若只有一个村庄内有宝物，或者所有村庄内都没有宝物，则输出 0。

样例

样例输入

4 5
1 2 30
2 3 50
2 4 60
2
3
4
2
1

样例输出

0
100
220
220
280

数据范围与提示

对于 $10\%$ 的数据，$1 \leq N \leq 100, 1 \leq M \leq 100$；

对于 $20\%$ 的数据，$1 \leq N \leq 1000, 1 \leq M \leq 1000$；

对于另外 $15\%$ 的数据，每个村庄最多成为两条道路的端点；

对于 $100\%$ 的数据，$1 \leq N \leq 100000,\ 1 \leq M \leq 100000,\ 1 \leq z \leq 10^9$。

题解

容易注意到，最优方案可以从 DFS 序最小的点开始，按 DFS 序递增的顺序访问每个点，然后再回到起点。也就是说，如果按 DFS 序对每个点排序，那么答案为

$$\sum_{i = 1}^n dis(P_i, P_{i + 1})$$

这里 $P_{n + 1} = P_1$。

这个想法是由虚树建树过程启发的。虚树的建树过程，同样也是按照 DFS 序排序，这样也能保证最后建出来的树形态正确。模拟一下就清楚了。

然后用 std::set 维护前继后继即可。

时间复杂度：$O(n \log n)$。

代码

#include <cstdio>
#include <set>

typedef long long int64;

#ifdef ONLINE_JUDGE
const int MAXN = 100000 + 10;
const int MAXLOG = 19;
#else
const int MAXN = 100 + 10;
const int MAXLOG = 5;
#endif

int n, m;

int pre[MAXN << 1], to[MAXN << 1], wdis[MAXN << 1], head[MAXN], wcnt;
int fa[MAXN][MAXLOG + 1];
int id[MAXN], dep[MAXN], idcnt;
int64 dis[MAXN];
bool vis[MAXN];

struct Cmp {
    bool operator()(const int x, const int y) { return id[x] < id[y]; }
};
typedef std::set<int, Cmp> Nodeset;
Nodeset set;

inline void addedge(int u, int v, int w) {
    wcnt++;
    pre[wcnt] = head[u];
    head[u] = wcnt;
    to[wcnt] = v;
    wdis[wcnt] = w;
}
inline void add2edge(int u, int v, int w) {
    addedge(u, v, w);
    addedge(v, u, w);
}

#define v to[i]
void dfs(int u) {
    int i;

    for (i = 1; fa[u][i - 1]; i++)
        fa[u][i] = fa[fa[u][i - 1]][i - 1];

    idcnt++;
    id[u] = idcnt;
    for (i = head[u]; i; i = pre[i]) {
        if (!id[v]) {
            dis[v] = dis[u] + wdis[i];
            dep[v] = dep[u] + 1;
            fa[v][0] = u;
            dfs(v);
        }
    }
}
#undef v

int lca(int x, int y) {
    int i;

    if (dep[x] < dep[y])
        std::swap(x, y);
    for (i = MAXLOG; ~i; i--)
        if (dep[fa[x][i]] >= dep[y])
            x = fa[x][i];
    if (x == y)
        return x;
    for (i = MAXLOG; ~i; i--) {
        if (fa[x][i] ^ fa[y][i]) {
            x = fa[x][i];
            y = fa[y][i];
        }
    }
    return fa[x][0];
}
inline int64 querydis(int x, int y) { return (x and y) ? (dis[x] + dis[y] - 2 * dis[lca(x, y)]) : 0; }

int main() {
#ifndef ONLINE_JUDGE
    freopen("2182.in", "r", stdin);
    freopen("2182.out", "w", stdout);
#endif

    Nodeset::iterator it;
    int i;
    int64 ans = 0;
    int x, y, z;

    scanf("%d %d", &n, &m);
    for (i = 1; i < n; i++) {
        scanf("%d %d %d", &x, &y, &z);
        add2edge(x, y, z);
    }
    dfs(1);

    for (i = 1; i <= m; i++) {
        scanf("%d", &x);

        y = z = 0;
        it = set.upper_bound(x);
        if (it != set.end())
            z = *it;
        if (it != set.begin()) {
            it--;
            if (vis[x]) {
                if (it != set.begin()) {
                    it--;
                    y = *it;
                }
            }
            else
                y = *it;
        }

        vis[x] = !vis[x];
        if (vis[x])
            set.insert(x);
        else
            set.erase(x);
        ans += (vis[x] ? 1 : -1) * (querydis(x, y) + querydis(x, z) - querydis(y, z));
        if (set.size() >= 2) {
            it = set.end();
            it--;
            printf("%lld\n", ans + querydis(*set.begin(), *it));
        }
        else
            printf("0\n");
    }
    return 0;
}
