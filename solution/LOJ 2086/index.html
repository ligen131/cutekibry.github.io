LOJ #2086. 「NOI2016」区间

内存限制：256 MiB 时间限制：3000 ms 标准输入输出
题目类型：传统 评测方式：文本比较
上传者：未知
原题链接 查看源代码 修改历史 版本对比 ::after

题目描述

在数轴上有 $n$ 个闭区间 $[l_1,r_1],[l_2,r_2],...,[l_n,r_n]$。现在要从中选出 $m$ 个区间，使得这 $m$ 个区间共同包含至少一个位置。换句话说，就是使得存在一个 $x$，使得对于每一个被选中的区间 $[l_i,r_i]$，都有 $l_i\leq x\leq r_i$。

对于一个合法的选取方案，它的花费为被选中的最长区间长度减去被选中的最短区间长度。区间 $[l_i,r_i]$ 的长度定义为 $r_i−l_i$，即等于它的右端点的值减去左端点的值。

求所有合法方案中最小的花费。如果不存在合法的方案，输出 $−1$。

输入格式

第一行包含两个正整数 $n,m$ 用空格隔开，意义如上文所述。保证 $1\leq m\leq n$。

接下来 $n$ 行，每行表示一个区间，包含用空格隔开的两个整数 $l_i$ 和 $r_i$ 为该区间的左右端点。

输出格式

只有一行，包含一个正整数，即最小花费。

样例

样例输入

6 3
3 5
1 2
3 4
2 2
1 5
1 4

样例输出

2

样例解释

如图，当 $n=6,~m=3$ 时，花费最小的方案是选取 $[3,5]$、$[3,4]$、$[1,4]$ 这三个区间，他们共同包含了 $4$ 这个位置，所以是合法的。其中最长的区间是 $[1,4]$，最短的区间是 $[3,4]$，所以它的花费是 $(4−1)−(4−3)=2$。

题解

注意到输入区间的顺序对方案没有影响，因此我们可以按区间长度排序。

这样一来，可以用类似双指针的方法做：

for (i = j = 1; i <= n; i++) {
    while (check(i, j) >= m) {
        ans = std::min(ans, (a[i].r - a[i].l) - (a[j].r - a[j].l));
        j++;
    }
}

这里 check(i, j) 表示选一个点，使得被第 $i, i + 1, \ldots, j$ 个区间包含的区间个数最多，这样的答案是多少，即

$$\text{check}(i, j) = \max_{x \in \mathbb{Z}} \{\sum_{k = i}^j [l_k \leq x \leq r_k]\}$$

可以用线段树维护。

时间复杂度：$O(n \log n)$。

代码

#include <algorithm>
#include <cstdio>

#define lp (p << 1)
#define rp (p << 1 | 1)
#define mid ((l + r) >> 1)
#define lson lp, l, mid
#define rson rp, mid + 1, r

const int MAXN = 1000000 + 10;
const int INF = 0x7fffffff;

struct Seg {
    int l, r;
    bool operator<(const Seg b) const { return r - l < b.r - b.l; }
};
Seg a[MAXN];
int posl[MAXN], posr[MAXN];
int num[MAXN], numn;

int n, m;
int maxv[MAXN << 2], mark[MAXN << 2];

inline void addnode(int p, int x) {
    maxv[p] += x;
    mark[p] += x;
}
inline void down(int p) {
    if (mark[p]) {
        addnode(lp, mark[p]);
        addnode(rp, mark[p]);
        mark[p] = 0;
    }
}
inline void up(int p) { maxv[p] = std::max(maxv[lp], maxv[rp]); }
void update(int p, int l, int r, int a, int b, int x) {
    if (a <= l and r <= b) {
        addnode(p, x);
        return;
    }
    down(p);
    if (a <= mid)
        update(lson, a, b, x);
    if (b> mid)
        update(rson, a, b, x);
    up(p);
}

int main() {
#ifndef ONLINE_JUDGE
    freopen("2086.in", "r", stdin);
    freopen("2086.out", "w", stdout);
#endif

    int i, j, ans = INF;

    scanf("%d %d", &n, &m);
    for (i = 1; i <= n; i++) {
        scanf("%d %d", &a[i].l, &a[i].r);
        numn++;
        num[numn] = a[i].l;
        numn++;
        num[numn] = a[i].r;
    }

    std::sort(a + 1, a + 1 + n);

    std::sort(num + 1, num + 1 + numn);
    for (i = 1; i <= n; i++) {
        posl[i] = std::lower_bound(num + 1, num + 1 + numn, a[i].l) - num;
        posr[i] = std::lower_bound(num + 1, num + 1 + numn, a[i].r) - num;
    }

    for (i = j = 1; i <= n; i++) {
        update(1, 1, numn, posl[i], posr[i], 1);
        while (maxv[1] >= m) {
            ans = std::min(ans, (a[i].r - a[i].l) - (a[j].r - a[j].l));
            update(1, 1, numn, posl[j], posr[j], -1);
            j++;
        }
    }
    if (ans ^ INF)
        printf("%d\n", ans);
    else
        printf("-1\n");
    return 0;
}
