ZJU #3329. One Person Game

内存限制：32 MiB 时间限制：1000 ms 标准输入输出
题目类型：传统 评测方式：文本比较
上传者：未知
原题链接 查看源代码 修改历史 版本对比 ::after

题目描述

有三个质地均匀的骰子，它们分别有 $K_1, K_2, K_3$ 个面。

轮流进行下列操作：

  1. 初始将计数器置零。
  2. 投掷三个骰子。如果第一个、第二个、第三个骰子朝上的数字分别为 $a, b, c$，将计数器置零，否则将计数器的值加上这些数字。
  3. 若计数器的值不超过 $n$，回到第二步；否则游戏结束。

请你计算投掷骰子的期望次数。

输入格式

多组数据。

第一行，一个整数 $T \: (0 < T \leq 300)$，表示数据组数。

接下来 $T$ 行，每行七个非负整数 $n, K_1, K_2, K_3, a, b, c \: (0 \leq n \leq 500, 1 < K_1, K_2, K_3 \leq 6, 1 \leq a \leq K_1, 1 \leq b \leq K_2, 1 \leq c \leq K_3)$。

输出格式

$T$ 行，每行一个实数，表示答案。你的输出和标准输出的误差必须不超过 $10^{-8}$。

样例

样例输入

2
0 2 2 2 1 1 1
0 6 6 6 1 1 1

样例输出

1.142857142857143
1.004651162790698

题解

设 $x_i$ 为计数器的值为 $i$ 时，结束游戏需要的期望步数；$p_k$ 为投掷 $k$ 分且不置零的概率；$p_0 = \frac 1{K_1K_2K_3}$，将计数器置零的概率。

显然，当 $i > n$ 时有 $x_i = 0$。

当 $i \leq n$ 时有

$$x_i = 1 + \left( \sum_{j = 1}^{K_1 + K_2 + K_3} p_j x_{i + j} \right) + p_0x_0$$

直接高斯消元是 $O(T(n^3 + nK + K^3))$ 的，通不过本题。

注意到很多状态和 $x_0$ 有关，因此考虑设

$$x_i = A(i)x_0 + B(i)$$

我们把 $x_i = A(i)x_0 + B(i)$ 代入到原式的 $x_{i + j}$ 中，得

$$ \begin{aligned} x_i &= 1 + \left( \sum_{j = 1}^{K_1 + K_2 + K_3} p_j x_{i + j} \right) + p_0x_0 \\ &= 1 + \left( \sum_{j = 1}^{K_1 + K_2 + K_3} p_j A(i + j) x_0 + p_j B(i + j) \right) + p_0x_0 \\ &= \left[ \left( \sum_{j = 1}^{K_1 + K_2 + K_3} p_j A(i + j) \right) + p_0 \right] x_0 + \left[ \left( \sum_{j = 1}^{K_1 + K_2 + K_3} p_j B(i + j) \right) + 1 \right] \\ \end{aligned} $$

因此可得 $A(i) = \left( \sum_{j = 1}^{K_1 + K_2 + K_3} p_j A(i + j) \right) + p_0, B(i) = \left(\sum_{j = 1}^{K_1 + K_2 + K_3}p_jB(i + j) \right) + 1$。

这样，就可以用 $O(nK)$ 的递推求出 $A(0), B(0)$，即可求出 $x_0 = \frac {B(i)}{1 - A(i)}$。

时间复杂度：$O(T(nK + K^3))$。

代码

#include <cstdio>
#include <cstring>

typedef long double extend;

const int MAXN = 550 + 10;

extend a[MAXN], b[MAXN];
extend p[MAXN];

int main() {
#ifndef ONLINE_JUDGE
    freopen("3329.in", "r", stdin);
    freopen("3329.out", "w", stdout);
#endif

    int T;
    int i, j, k;
    int n, n1, n2, n3, m1, m2, m3;

    scanf("%d", &T);
    while (T) {
        memset(a, 0, sizeof(a));
        memset(b, 0, sizeof(b));
        memset(p, 0, sizeof(p));

        scanf("%d %d %d %d %d %d %d", &n, &n1, &n2, &n3, &m1, &m2, &m3);

        p[0] = 1.0 / n1 / n2 / n3;
        for (i = 1; i <= n1; i++)
            for (j = 1; j <= n2; j++)
                for (k = 1; k <= n3; k++)
                    p[i + j + k] += p[0];
        p[m1 + m2 + m3] -= p[0];
        for (i = n; ~i; i--) {
            for (j = 1; j <= n1 + n2 + n3; j++) {
                a[i] += p[j] * a[i + j];
                b[i] += p[j] * b[i + j];
            }
            a[i] += p[0];
            b[i] += 1;
        }

        printf("%.15Lf\n", b[0] / (1 - a[0]));
        T--;
    }
    return 0;
}
