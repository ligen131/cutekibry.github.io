LOJ #2038. 「SHOI2015」超能粒子炮・改

内存限制：256 MiB 时间限制：1000 ms 标准输入输出
题目类型：传统 评测方式：文本比较
上传者：未知
原题链接 查看源代码 修改历史 版本对比 ::after

题目描述

曾经发明了脑洞治疗仪与超能粒子炮的发明家 SHTSC 又公开了他的新发明：超能粒子炮・改——一种可以发射威力更加强大的粒子流的神秘装置。

超能粒子炮・改相比超能粒子炮，在威力上有了本质的提升。它有两个参数 $n$、$k$，它会向每个编号为 $0$ 到 $k$（包含两端）的位置 $i$ 发射威力为 $\mathrm{C}_n^i \mathbin{\mathrm{mod}} 2333$ 的粒子流。

现在 SHTSC 给出了他的超能粒子炮・改的参数，让你求出其发射的粒子流的威力之和除以 $2333$ 所得的余数。

输入格式

第一行一个整数 $t$ 表示数据组数。

之后 $t$ 行，每行两个整数 $n$、$k$，含义如题面描述。

输出格式

$t$ 行，每行一个整数，表示其粒子流的威力之和模 $2333$ 的值。

样例

样例输入

3
5 5
10 7
1145 14

样例输出

32
968
763

数据范围与提示

对于 $10\%$ 的数据，$t = 1$，$n, k \leq 1000$；

对于 $30\%$ 的数据，$t = 1$，$n, k \leq 1000000$；

对于 $50\%$ 的数据，$t = 1$，$n \leq 10^{18}, k \leq 1000$；

对于 $70\%$ 的数据，$t = 100$，$n, k \leq 10^{18}$；

对于 $100\%$ 的数据，$t = 100000$，$n, k \leq 10^{18}$。

题解

首先，我们要知道 Lucas 定理。

Lucas 定理：对于任意质数 $p \in \mathbb{P}$ 和非负整数 $n, m$ ，有：

$$ \mathrm{C}^n_m \equiv \mathrm{C}^{n \setminus p}_{m \setminus p} \mathrm{C}^{n \bmod p}_{m \bmod p} \pmod p $$

考虑设 $f(n, k) = \sum_{i=0}^k \mathrm{C}^i_n$ 。

为了方便，下文中均保证 $k=ap-1 | a \in \mathbb{N^*}$ 。

由于 Lucas 定理中存在整除和取模，我们想到类似分段的求法：

$$ f(n, k) = \sum_{i=0}^k \mathrm{C}^i_n \bmod p \\ f(n, k) = \sum_{i=0}^k \mathrm{C}^{i \setminus p}_{n \setminus p} \mathrm{C}^{i \bmod p}_{n \bmod p} \bmod p \\ f(n, k) = \sum_{i=0}^{k \setminus p} \sum_{j=0}^{\min\{k-ip, p-1\}} \mathrm{C}^i_{n \setminus p} \mathrm{C}^j_{n \bmod p} \bmod p \\ $$

直白点写，就是：

$$ f(n, k) = (\mathrm{C}^0_{n \setminus p} \sum_{i=0}^{p-1} C^i_{n \bmod p} + \mathrm{C}^1_{n \setminus p} \sum_{i=0}^{p-1} C^i_{n \bmod p} + \ldots) \bmod p $$

提取公因式 $\sum_{i=0}^{p-1} C^i_{n \bmod p}$ ，得：

$$ f(n, k) = (\sum_{i=0}^{p-1} C^i_{n \bmod p}) (\mathrm{C}^0_{n \setminus p} + \mathrm{C}^1_{n \setminus p} + \ldots + \mathrm{C}^{k \setminus p-1}_{n \setminus p}) + (\sum_{i=0}^{k \bmod p} C^i_{n \bmod p}) (\mathrm{C}^{k \setminus p}_{n \setminus p}) \bmod p\\ f(n, k) = (\sum_{i=0}^{p-1} C^i_{n \bmod p}) (\sum_{i=0}^{k \setminus p-1} \mathrm{C}^i_{n \setminus p}) + (\sum_{i=0}^{k \bmod p} C^i_{n \bmod p})(\mathrm{C}^{k \setminus p}_{n \setminus p}) \bmod p \\ f(n, k) = f(n \bmod p, p-1) f(n \setminus p, k \setminus p-1) + f(n \bmod p, k \bmod p) (\mathrm{C}^{k \setminus p}_{n \setminus p}) \bmod p $$

这样的复杂度是多少呢？

对于 $f(n \bmod p, p-1)$ ，可以 $O(p^2)$ 预处理出 $\{\mathrm{C}^n_m|n, m \in [0, p-1]\}$ ， $O(1)$ 求出。

对于 $f(n \setminus p, k \setminus p-1)$ ，每次 $k, p$ 都除以 $p$ ，递归次数是 $\log_p n$ 的。

对于 $f(n \bmod p, k \bmod p)(\mathrm{C}^{k \setminus p}_{n \setminus p})$ ，左边的部分同理 $O(1)$ 求出，右边的部分 $O(\log_p n)$ 求出。

设 $f(n, k)$ 的复杂度为 $T(n, k)$ ，那么：

$$ T(n, k)= \begin{cases} 1, &\max\{n, k\} < p \\ 1 + T(k \setminus p, n \setminus p) + \log_p n, &\text{otherwise} \end{cases} $$

因为递归最多计算 $\log_p n$ 个 $T(n, k)$ ，每个 $T(n, k)$ 需要 $\log_p n+1$ 的时间计算， 所以 $T(n, k) = O(\log_p n) \times O(\log_p n+1) = O(\log_p^2 n)$ 。

总时间复杂度为 $O(p^2+t\log_p^2 n)$ 。

代码

#include <cstdio>

const int P = 2333;

typedef long long int64;

int pref[P][P];
int fact[P], inv[P];
inline int C(int64 n, int64 m) {
    if (n > m)
        return 0;
    if (n < P and m < P)
        return fact[m] * inv[fact[n]] % P * inv[fact[m - n]] % P;
    else
        return C(n % P, m % P) * C(n / P, m / P) % P;
}
inline int f(int64 n, int64 k) {
    if (k > n)
        k = n;
    if (k < 0)
        return 0;
    else if (n < P and k < P)
        return pref[n][k];
    else
        return (f(n % P, P - 1) * f(n / P, k / P - 1) +
                f(n % P, k % P) * C(k / P, n / P)) % P;
}
inline void prepare() {
    int i, j;

    fact[0] = fact[1] = 1;
    inv[1] = 1;
    for (i = 2; i < P; i++) {
        inv[i] = (P - P / i) * inv[P % i] % P;
        fact[i] = i * fact[i - 1] % P;
    }

    for (i = 0; i < P; i++) {
        pref[i][0] = C(0, i);
        for (j = 1; j < P; j++) {
            pref[i][j] = C(j, i) + pref[i][j - 1];
            if (pref[i][j] >= P)
                pref[i][j] -= P;
        }
    }
}

int main() {
    #ifndef ONLINE_JUDGE
    freopen("2038.in", "r", stdin);
    freopen("2038.out", "w", stdout);
    #endif

    int t;
    int64 n, k;

    prepare();

    scanf("%d", &t);
    while (t) {
        scanf("%lld %lld", &n, &k);
        printf("%d\n", f(n, k));
        t--;
    }
    return 0;
}
