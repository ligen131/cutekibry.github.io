LOJ #2056. 「TJOI / HEOI2016」序列

内存限制：256 MiB 时间限制：1000 ms 标准输入输出
题目类型：传统 评测方式：文本比较
上传者：未知
原题链接 查看源代码 修改历史 版本对比 ::after

题目描述

佳媛姐姐过生日的时候，她的小伙伴从某宝上买了一个有趣的玩具送给他。

玩具上有一个数列，数列中某些项的值可能会变化，但同一个时刻最多只有一个值发生变化。现在佳媛姐姐已经研究出了所有变化的可能性，她想请教你，能否选出一个子序列，使得在任意一种变化中，这个子序列都是不降的？请你告诉她这个子序列的最长长度即可。

输入格式

输入的第一行有两个正整数 $ n, m $，分别表示序列的长度和变化的个数。

接下来一行有 $ n $ 个数，表示这个数列原始的状态。

接下来 $ m $ 行，每行有两个数 $ x, y $，表示数列的第 $ x $ 项可以变化成 $ y $ 这个值。

输出格式

输出一个整数，表示对应的答案。

样例

样例输入

3 4
1 2 3
1 2
2 3
2 1
3 4

样例输出

3

样例解释

注意：每种变化最多只有一个值发生变化。

在样例输入中，所有的变化是：

1 2 3
2 2 3
1 3 3
1 1 3
1 2 4

选择子序列为原序列，即在任意一种变化中均为不降子序列。

数据范围与提示

对于所有的数据，$1 \leq x \leq n$，所有数字均为正整数，且小于等于 $100000$。

题解

设 $A_i, L_i, R_i$ 分别为序列第 $i$ 个数原来的数值、能变化为的最小数值、能变化为的最大数值，那么选出的序列必须满足：

  • $A_i \leq L_{i+1}$
  • $A_i \geq R_{i-1}$

设 $f[i]$ 表示以 $A_i$ 结尾的子序列最长长度，那么有

$$f[i]=\max\{f[j]|1 \leq j < i,A_j \leq L_i,R_{j} \leq A_i\}+1$$

CDQ 分治一下即可。

当然用主席树之类的...应该也可以？

时间复杂度：$O(n \log^2 n)$。

懒得算具体到底是多少了，所以假定 $n, A_i, x$ 同阶。

代码

#include <algorithm>
#include <cstdio>

#ifdef ONLINE_JUDGE
const int MAXN = 100000 + 10;
#else
const int MAXN = 1000;
#endif

struct Node {
    int p, a, l, r;
};
Node a[MAXN];
int f[MAXN];
int c[MAXN];
int n, m;

inline int lowbit(int x) { return x & -x; }
inline void update(int p, int x) {
    for (; p < MAXN; p += lowbit(p))
        c[p] = std::max(c[p], x);
}
inline void clear(int p) {
    for (; p < MAXN; p += lowbit(p))
        c[p] = 0;
}
inline int query(int p) {
    int res = 0;
    for (; p; p ^= lowbit(p))
        res = std::max(res, c[p]);
    return res;
}

bool cmpbya(Node x, Node y) { return x.a < y.a; }
bool cmpbyl(Node x, Node y) { return x.l < y.l; }
bool cmpbyp(Node x, Node y) { return x.p < y.p; }

void cdq(int l, int r) {
    int mid = (l + r) >> 1, i, j, k;

    if (l == r) {
        if (!f[a[l].p])
            f[a[l].p] = 1;
        return;
    }
    std::sort(a + l, a + r + 1, cmpbyp);
    cdq(l, mid);
    std::sort(a + l, a + mid + 1, cmpbya);
    std::sort(a + mid + 1, a + r + 1, cmpbyl);

    j = l;
    for (i = mid + 1; i <= r; i++) {
        while (j <= mid and a[j].a <= a[i].l) {
            update(a[j].r, f[a[j].p]);
            j++;
        }
        f[a[i].p] = std::max(f[a[i].p], query(a[i].a) + 1);
    }
    for (i = l; i < j; i++)
        clear(a[i].r);

    cdq(mid + 1, r);
}

int main() {
#ifndef ONLINE_JUDGE
    freopen("2056.in", "r", stdin);
    freopen("2056.out", "w", stdout);
#endif

    int i, x, y;
    int ans = 0;

    scanf("%d %d", &n, &m);
    for (i = 1; i <= n; i++) {
        a[i].p = i;
        scanf("%d", &a[i].a);
        a[i].l = a[i].r = a[i].a;
    }
    for (i = 1; i <= m; i++) {
        scanf("%d %d", &x, &y);
        a[x].l = std::min(a[x].l, y);
        a[x].r = std::max(a[x].r, y);
    }

    cdq(1, n);

    for (i = 1; i <= n; i++)
        ans = std::max(ans, f[i]);
    printf("%d\n", ans);
    return 0;
}
