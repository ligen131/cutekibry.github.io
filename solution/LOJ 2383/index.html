LOJ #2383. 「HNOI2013」游走

内存限制：256 MiB 时间限制：1000 ms 标准输入输出
题目类型：传统 评测方式：文本比较
上传者：未知
原题链接 查看源代码 修改历史 版本对比 ::after

题目描述

一个无向连通图，顶点从 $1$ 编号到 $N$ ，边从 $1$ 编号到 $M$ 。

小 Z 在该图上进行随机游走，初始时小 Z 在 $1$ 号顶点，每一步小 Z 以相等的概率随机选择当前顶点的某条边，沿着这条边走到下一个顶点，获得等于这条边的编号的分数。当小 Z 到达 $N$ 号顶点时游走结束，总分为所有获得的分数之和。 现在，请你对这 $M$ 条边进行编号，使得小 Z 获得的总分的期望值最小。

输入格式

输入第一行是正整数 $N$ 和 $M$ ，分别表示该图的顶点数和边数，接下来 $M$ 行每行是整数 $u , v$ ($1 \le u,v \le N$)，表示顶点 $u$ 与顶点 $v$ 之间存在一条边。

输出格式

输出仅包含一个实数，表示最小的期望值，保留 3 位小数。

样例

样例输入

3 3
2 3
1 2
1 3

样例输出

3.333

数据范围与提示

对于 $30\%$ 的数据， $N \le 10$ 。

对于 $100\%$ 的数据， $2 \le N \le 500$ 。

输入数据保证是一个无向简单连通图。

数据根据 cyand1317 的 std 重制。

题解

显然，有一个贪心的想法：让期望经过多的边编号小。

怎么计算边经过的期望次数？

设边方程是比较困难的，我们不如设点方程。

设 $x_p$ 为从点 $p$ 出发经过点数的期望，$deg_p$ 为点 $p$ 的度，那么有

$$x_u = \sum \frac{x_v}{deg_v}$$

其中 $u$ 能到达点 $v$。因为从 $v$ 点出发，有 $\frac 1{deg_v}$ 的概率会走回来。

显然，我们有 $x_n = 0$（因为不能从点 $n$ 出发），还有 $x_1 = 1 + \sum \frac {x_v}{deg_v}$（因为我们从点 $1$ 出发）。

利用高斯消元解方程组后，经过边 $u \rightarrow v$ 的期望次数即为 $\frac {x_u}{deg_u} + \frac {x_v}{deg_v}$，因为经过边 $u \rightarrow v$ 只有两种可能，一种从 $u$ 走到 $v$，概率为 $deg_u$；另一种从 $v$ 走到 $u$，概率为 $deg_v$。

时间复杂度：$O(n^3)$。

代码

#include <algorithm>
#include <cstdio>

#ifdef ONLINE_JUDGE
const int MAXN = 500 + 10;
const int MAXM = 250000 + 10;
#else
const int MAXN = 30;
const int MAXM = 900;
#endif

double a[MAXN][MAXN];
double x[MAXN];
double t[MAXM];
int deg[MAXN];
int u[MAXM], v[MAXM];
int n, m;

void solve() {
    int i, j, k;
    double p;

    for (i = 1; i < n; i++) {
        if (a[i][i] == 0) {
            j = i + 1;
            while (a[j][i] == 0 and j <= n)
                j++;
            for (k = i; k <= n + 1; k++)
                std::swap(a[i][k], a[j][k]);
        }
        for (j = i + 1; j <= n; j++) {
            p = a[j][i] / a[i][i];
            for (k = i; k <= n + 1; k++)
                a[j][k] = a[j][k] - a[i][k] * p;
        }
    }

    x[n] = a[n][n + 1];
    for (i = n - 1; i; i--) {
        for (j = 1; j <= n; j++)
            a[j][n + 1] -= a[j][i + 1] * x[i + 1];
        x[i] = a[i][n + 1] / a[i][i];
    }
}

int main() {
#ifndef ONLINE_JUDGE
    freopen("2383.in", "r", stdin);
    freopen("2383.out", "w", stdout);
#endif

    int i;
    double ans = 0;

    scanf("%d %d", &n, &m);
    for (i = 1; i <= m; i++) {
        scanf("%d %d", &u[i], &v[i]);
        deg[u[i]]++;
        deg[v[i]]++;
    }
    for (i = 1; i <= m; i++) {
        if (u[i] ^ n)
            a[u[i]][v[i]] = -1.0 / deg[v[i]];
        if (v[i] ^ n)
            a[v[i]][u[i]] = -1.0 / deg[u[i]];
    }

    for (i = 1; i <= n; i++)
        a[i][i] = 1;
    a[1][n + 1] = 1;
    solve();

    for (i = 1; i <= m; i++)
        t[i] = x[u[i]] / deg[u[i]] + x[v[i]] / deg[v[i]];
    std::sort(t + 1, t + 1 + m);
    for (i = 1; i <= m; i++)
        ans += t[m - i + 1] * i;
    printf("%.3lf\n", ans);
    return 0;
}
