LOJ #2245. 「NOI2014」魔法森林

内存限制：256 MiB 时间限制：1000 ms 标准输入输出
题目类型：传统 评测方式：文本比较
上传者：未知
原题链接 查看源代码 修改历史 版本对比 ::after

题目描述

为了得到书法大家的真传，小 E 同学下定决心去拜访住在魔法森林中的隐士。

魔法森林可以被看成一个包含 $N$ 个节点 $M$ 条边的无向图，节点标号为 $1,\ldots,N$，边标号为 $1,\ldots,M$。初始时小 E 同学在号节点 $1$，隐士则住在 $N$ 号节点。小 E 需要通过这一片魔法森林，才能够拜访到隐士。

魔法森林中居住了一些妖怪。每当有人经过一条边的时候，这条边上的妖怪就会对其发起攻击。幸运的是，在 $1$ 号节点住着两种守护精灵：$A$ 型守护精灵与 $B$ 型守护精灵。小 E 可以借助它们的力量，达到自己的目的。

只要小 E 带上足够多的守护精灵，妖怪们就不会发起攻击了。具体来说，无向图中的每一条边 $E_i$ 包含两个权值 $A_i$ 与 $B_i$。若身上携带的 A 型守护精灵个数不少于 $A_i$，且 $B$ 型守护精灵个数不少于 $B_i$，这条边上的妖怪就不会对通过这条边的人发起攻击。当且仅当通过这片魔法森林的过程中没有任意一条边的妖怪向小 E 发起攻击，他才能成功找到隐士。

由于携带守护精灵是一件非常麻烦的事，小 E 想要知道，要能够成功拜访到隐士，最少需要携带守护精灵的总个数。守护精灵的总个数为 $A$ 型守护精灵的个数与 $B$ 型守护精灵的个数之和。

输入格式

第一行包含两个整数 $N,M$，表示无向图共有 $N$ 个节点，$M$ 条边。

接下来 $M$ 行，第 $i$ 行包含四个正整数 $X_i,Y_i,A_i,B_i$，描述第 $i$ 条无向边。其中 $X_i$ 与 $Y_i$ 为该边两个端点的标号，$A_i$ 与 $B_i$ 的含义如题所述。

注意：数据中可能包含重边与自环。

输出格式

输出一行一个整数：如果小 E 可以成功拜访到隐士，输出小 E 最少需要携带的守护精灵的总个数；

如果无论如何小 E 都无法拜访到隐士，输出 “-1”（不含引号）。

样例

样例输入 1

4 5
1 2 19 1
2 3 8 12
2 4 12 15
1 3 17 8
3 4 1 17

样例输出 1

32

样例解释 1

如果小 E 走路径 $1\rightarrow2\rightarrow4$，需要携带 $19+15=34$ 个守护精灵；
如果小 E 走路径 $1\rightarrow3\rightarrow4$，需要携带 $17+17=34$ 个守护精灵；
如果小 E 走路径 $1\rightarrow2\rightarrow3\rightarrow4$，需要携带 $19+17=36$ 个守护精灵；
如果小 E 走路径 $1\rightarrow3\rightarrow2\rightarrow4$，需要携带 $17+15=32$ 个守护精灵。
综上所述，小 E 最少需要携带 $32$ 个守护精灵。

样例输入 2

3 1
1 2 1 1

样例输出 2

-1

样例解释 2

小 E 无法从 $1$ 号节点到达 $3$ 号节点，故输出 -1。

数据范围与提示

对所有的数据，$2 \leq n \leq 50000,\ 0 \leq m \leq 100000,\ 1 \leq a_i ,b_i \leq 50000$。

题解

$a, b$ 两维太复杂了。我们可以考虑对边按照 $a_i$ 排序，并对于每个 $i$，将 $a_j \leq a_i$ 的所有边加入该图。在这种情况下，最小答案即为 $a_i$ 加上节点 $1$ 到 $n$ 的最小瓶颈路。

最小瓶颈路即为最小生成树上的路径。

也就是说，我们需要写一个数据结构，支持动态维护最小生成树。

显然，最小生成树是一棵树。删边加边的问题，可以考虑用 Link-Cut Tree 解决。

假定在原来的树上，存在简单路径 $u \rightarrow p_1 \rightarrow p_2 \rightarrow \ldots \rightarrow p_k \rightarrow v$。现在我们要加入边 $u \rightarrow v$，权值为 $w$，如何维护最小生成树。

可以贪心：找出该路径上权值最大的边，若该边权值大于 $w$，则删去该边，并加入边 $u \rightarrow v$。

也就是维护一颗 Link-Cut Tree，支持加边、删边，并需要维护最大边权的编号。

由于边权问题我们不是很能做，因此我们可以把第 $i$ 条边 $u \rightarrow v$ 拆成三个点 $u, v, i + n$。转化后是一个点权问题，可以轻松 Link-Cut Tree 维护。

时间复杂度：$O(n \log n)$。

代码

#include <algorithm>
#include <cstdio>

#ifdef ONLINE_JUDGE
const int MAXN = 200000 + 10;
#else
const int MAXN = 1000 + 10;
#endif

struct Edge {
    int u, v, a, b;
    bool operator<(const Edge e) const { return a < e.a; }
};
Edge e[MAXN];

int n, m;

int maxp[MAXN], val[MAXN], from[MAXN], to[MAXN], nodecnt;
bool mark[MAXN];
int ch[MAXN][2], fa[MAXN];

#define lc ch[x][0]
#define rc ch[x][1]
#define clr fa[0] = ch[0][0] = ch[0][1] = val[0] = maxp[0] = 0
inline bool isroot(int x) { return ch[fa[x]][0] ^ x and ch[fa[x]][1] ^ x; }
inline bool identify(int x) { return ch[fa[x]][0] ^ x; }
inline int getmaxp(int x, int y) { return val[x] > val[y] ? x : y; }
inline void revnode(int x) {
    mark[x] = !mark[x];
    std::swap(lc, rc);
}
inline void up(int x) { maxp[x] = getmaxp(x, getmaxp(maxp[lc], maxp[rc])); }
inline void down(int x) {
    if (mark[x]) {
        if (lc)
            revnode(lc);
        if (rc)
            revnode(rc);
        mark[x] = false;
    }
}
inline void pushdown(int x) {
    static int s[MAXN];
    int stop = 0;

    while (!isroot(x)) {
        stop++;
        s[stop] = x;
        x = fa[x];
    }
    down(x);
    while (stop) {
        down(s[stop]);
        stop--;
    }
}
inline void rotate(int x) {
    bool d = identify(x);
    int y = fa[x], z = ch[x][!d];

    fa[x] = fa[y];
    if (!isroot(y))
        ch[fa[y]][identify(y)] = x;
    fa[y] = x;
    ch[x][!d] = y;
    fa[z] = y;
    ch[y][d] = z;
    up(y);
    up(x);
    clr;
}
inline void splay(int x) {
    pushdown(x);
    while (!isroot(x)) {
        if (!isroot(fa[x]) and identify(x) == identify(fa[x]))
            rotate(fa[x]);
        rotate(x);
    }
}
inline void access(int x) {
    int y;
    for (y = 0; x; y = x, x = fa[x]) {
        splay(x);
        rc = y;
        up(x);
    }
}
inline void makeroot(int x) {
    access(x);
    splay(x);
    revnode(x);
}
inline int findroot(int x) {
    access(x);
    splay(x);
    while (down(x), lc)
        x = lc;
    return x;
}
inline void split(int x, int y) {
    makeroot(x);
    access(y);
    splay(y);
}

inline int querypath(int x, int y) {
    split(x, y);
    return maxp[y];
}
inline void link(int x, int y) {
    makeroot(x);
    makeroot(y);
    fa[x] = y;
}
inline void cut(int x, int y) {
    split(x, y);
    fa[x] = ch[y][0] = 0;
    up(y);
}
#undef lc
#undef rc

int main() {
#ifndef ONLINE_JUDGE
    freopen("2245.in", "r", stdin);
    freopen("2245.out", "w", stdout);
#endif

    int i, u, v, a, b, p;
    int ans = 0x7fffffff;
    bool flag;

    scanf("%d %d", &n, &m);
    nodecnt = n;
    for (i = 1; i <= m; i++)
        scanf("%d %d %d %d", &e[i].u, &e[i].v, &e[i].a, &e[i].b);

    std::sort(e + 1, e + 1 + m);

    for (i = 1; i <= m; i++) {
        u = e[i].u;
        v = e[i].v;
        a = e[i].a;
        b = e[i].b;
        flag = true;

        if (findroot(u) == findroot(v)) {
            p = querypath(u, v);
            if (val[p] > b) {
                cut(e[p - n].u, p);
                cut(e[p - n].v, p);
            }
            else
                flag = false;
        }

        if (flag) {
            val[i + n] = b;
            link(u, i + n);
            link(v, i + n);
        }

        if (findroot(1) == findroot(n))
            ans = std::min(ans, a + val[querypath(1, n)]);
    }
    printf("%d\n", (ans ^ 0x7fffffff) ? ans : -1);
    return 0;
}
