LOJ #2993. 「AHOI2018 初中组」根式化简

内存限制：256 MiB 时间限制：1000 ms 标准输入输出
题目类型：传统 评测方式：文本比较
上传者：未知
原题链接 查看源代码 修改历史 版本对比 ::after

题目描述

小可可在学习「立方根」的知识时碰到了这样的问题。

将下面根式化简为最简根式：

  1. $\sqrt[3]{125}$
  2. $\sqrt[3]{81}$
  3. $\sqrt[3]{52}$

这个问题对于小可可来说太简单了，他很快就算出了答案：

  1. $5$
  2. $3\sqrt[3]{3}$
  3. $\sqrt[3]{52}$

小可可知道任意形如 $\sqrt[3]{x}$ 的根式，化简后一定可以被写成形如 $a\sqrt[3]{b}$ ​的最简根式。他觉得这很有趣，就仿照出了不少题，但没一会儿就被密密麻麻的根式绕晕了，于是他向你求助：

给定 $n$ 个形如 $\sqrt[3]{x}$ 的根式，请你将它们化简为形如 $a\sqrt[3]{b}$ 的最简形式，为了方便，你只需要输出其中的 $a$ 即可。

如果你没有学过这部分数学知识，你可以认为题意是：给你 $n$ 个正整数 $x$，对于每一个 $x$，你需要求出整数 $a,b$ 使得 $a^3 \times b = x$，输出最大的整数 $a$ 即可。

输入格式

输入有两行：

第一行一个整数 $n$，表示有 $n$ 个形如 $\sqrt[3]{x}$ 的根式；

第二行 $n$ 个正整数，依次给出每个 $x$。

输出格式

输出 $n$ 行，每行一个正整数，第 $i$ 行正整数表示你对输入中第 $i$ 个 $x$ 给出的答案。

样例

样例输入

3
125 81 52

样例输出

5
3
1

数据范围与提示

测试点编号 $n$ $x$
$1,2$ $n\le 10$ $x\le 10^6$
$3,4$ $n\le 10$ $x\le 10^9$
$5,6$ $n\le 100$ $x\le 10^{18}$ 且 $x$ 为完全立方数
$7,8$ $n\le 500$ $x\le 10^{18}$
$9,10$ $n\le 10^4$ $x\le 10^{18}$

题解

首先，最简单的方法，预处理出所有 $\sqrt[3]{10^{18}} = 10^6$ 以内的素数，然后分解因数。$10^6$ 以内的素数共有 $78498$ 个，超时。

注意到，对于一些比较大的素数立方，我们可以用较小的素数去分解它，然后直接判断即可。例如 $P = 30559, \lfloor \frac n{P^3} \rfloor = \lfloor \frac {10^{18}}{28537597966879} = 31688$，那么我们只需要对 $x$ 用 $31688$ 以内的素数进行分解，然后判断 $x$ 是否等于 $P^3$ 即可。

具体地说，我们枚举前 $B$ 个素数，并判断 $x$ 是否为它们立方的倍数。对于剩下的素数就可以用上面的方法，枚举 $\lfloor \frac x{P_B^3} \rfloor$ 以内的素数，对 $x$ 进行分解，然后判断 $x$ 是否等于 $P^3$ 即可。

判断的方式可以直接用二分。

时间复杂度：$O(n(B + \pi(\lfloor \frac x{P_B^3} \rfloor) + \log \pi(\sqrt[3] x)) + P_B + \lfloor \frac x{P_B^3} \rfloor + \sqrt[3]x)$，$\pi(n)$ 表示不超过 $n$ 的素数个数。

可以选择 $B = 3400$，$\pi(\lfloor \frac n{P_B^3} \rfloor) = \pi(\lfloor \frac {10^{18}}{31601^3} \rfloor) = \pi(31688) = 3408$。

代码

#include <algorithm>
#include <cstdio>

const int MAXP = 31700;
const int MAXB = 1000000 + 10;

typedef long long int64;

int p[MAXB], pn;
int64 p3[MAXB];
bool notp[MAXB];

int main() {
#ifndef ONLINE_JUDGE
    freopen("2993.in", "r", stdin);
    freopen("2993.out", "w", stdout);
#endif

    int i, j;
    int n;
    int64 x;
    int ans;

    notp[1] = true;
    for (i = 2; i < MAXB; i++) {
        if (!notp[i]) {
            pn++;
            p[pn] = i;
            p3[pn] = 1ll * i * i * i;
            for (j = i * 2; j < MAXB; j += i)
                notp[j] = true;
        }
    }

    scanf("%d", &n);
    while (n) {
        ans = 1;

        scanf("%lld", &x);
        for (i = 1; p[i] <= MAXP; i++) {
            while (!(x % p3[i])) {
                x /= p3[i];
                ans *= p[i];
            }
            while (!(x % p[i]))
                x /= p[i];
        }
        i = std::lower_bound(p3 + 1, p3 + 1 + pn, x) - p3;
        if (p3[i] == x)
            ans *= p[i];
        printf("%d\n", ans);
        n--;
    }
    return 0;
}
