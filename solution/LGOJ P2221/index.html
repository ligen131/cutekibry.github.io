LGOJ #P2221. 「HAOI2012」高速公路

上传者：未知

题目描述

Y901 高速公路是一条重要的交通纽带，政府部门建设初期的投入以及使用期间的养护费用都不低，因此政府在这条高速公路上设立了许多收费站。

Y901 高速公路是一条由 N-1 段路以及 N 个收费站组成的东西向的链，我们按照由西向东的顺序将收费站依次编号为 1~N，从收费站 i 行驶到 i+1(或从 i+1 行驶到 i) 需要收取 Vi 的费用。高速路刚建成时所有的路段都是免费的。

政府部门根据实际情况，会不定期地对连续路段的收费标准进行调整，根据政策涨价或降价。

无聊的小 A 同学总喜欢研究一些稀奇古怪的问题，他开车在这条高速路上行驶时想到了这样一个问题: 对于给定的 l,r(l<r), 在第 l 个到第 r 个收费站里等概率随机取出两个不同的收费站 a 和 b，那么从 a 行驶到 b 将期望花费多少费用呢?

输入格式

第一行 2 个正整数 N,M，表示有 N 个收费站，M 次调整或询问

接下来 M 行，每行将出现以下两种形式中的一种

C l r v 表示将第 l 个收费站到第 r 个收费站之间的所有道路的通行费全部增加 v

Q l r 表示对于给定的 l,r，要求回答小 A 的问题

所有 C 与 Q 操作中保证 1<=l<r<=N

输出格式

对于每次询问操作回答一行，输出一个既约分数

若答案为整数 a，输出 a/1

样例

样例输入

4 5
C 1 4 2
C 1 2 -1
Q 1 2
Q 2 4
Q 1 4

样例输出

1/1
8/3
17/6

数据范围与提示

所有 C 操作中的 v 的绝对值不超过 10000

在任何时刻任意道路的费用均为不超过 10000 的非负整数

题解

我们考虑用线段树维护。

设节点 $p$ 代表区间 $[L, R]$ ， $fsum[p]$ 为该区间的所有子区间的区间和。

也就是说：

$$ fsum[p]=\sum_{i=L}^R \sum_{j=i}^R \sum_{k=i}^j A_k $$

考虑怎么通过代表区间 $[L, mid]$ 的节点 $p1$ 和代表区间 $[mid+1, R]$ 的节点 $p2$ 的信息获得节点 $p$ 的信息。

注意到所有的子区间可以分解为三种类型：

  1. 区间在左节点内，即区间 $[l, r]$ 满足 $L\leq l \leq r \leq mid$ ；
  2. 区间在右节点内，即区间 $[l, r]$ 满足 $mid < l \leq r \leq R$ ；
  3. 区间跨越了中间的点，即区间 $[l, r]$ 满足 $L \leq l \leq mid, mid < r \leq R$ 。

前两种类型就是 $fsum[p1]+fsum[p2]$ 。至于第三种类型，写出来是这样的：

$$ \sum_{i=L}^{mid} \sum_{j=mid+1}^R \sum_{k=i}^j A_k $$

接下来，让我们推一下这个的公式。

（注：下面这个式子和上面的相同，为了直观换成了这种比较容易理解的方式。）

$$ \begin{aligned} &((A_L+A_{L+1}+\ldots+A_{mid})+(A_{mid+1}))+((A_L+A_{L+1}+\ldots+A_{mid})+(A_{mid+1}+A_{mid+2}))+\ldots+(A_L+A_{L+1}+\ldots+A_R) \\ +&((A_{L+1}+A_{L+2}+\ldots+A_{mid})+(A_{mid+1}))+((A_{L+1}+A_{L+2}+\ldots+A_{mid})+(A_{mid+1}+A_{mid+2}))+\ldots+(A_{L+1}+A_{L+2}+\ldots+A_R) \\ &\vdots \\ +&((A_{mid})+(A_{mid+1}))+((A_{mid})+(A_{mid+1}+A_{mid+2}))+\ldots+(A_{mid}+A_{mid+1}+\ldots+A_R) \\ \end{aligned} $$

我们将左边一半和右边一半拆出来，变成：

$$ \begin{aligned} &(R-mid)(A_L+A_{L+1}+\ldots+A_{mid})+((A_{mid+1})+(A_{mid+1}+A_{mid+2})+\ldots+(A_{mid+1}+A_{mid+2}+\ldots+A_R)) \\ +&(R-mid)(A_{L+1}+A_{L+2}+\ldots+A_{mid})+((A_{mid+1})+(A_{mid+1}+A_{mid+2})+\ldots+(A_{mid+1}+A_{mid+2}+\ldots+A_R)) \\ &\vdots \\ +&(R-mid)(A_{mid})+((A_{mid+1})+(A_{mid+1}+A_{mid+2})+\ldots+(A_{mid+1}+A_{mid+2}+\ldots+A_R)) \\ =&(R-mid)((A_{mid})+(A_{mid}+A_{mid-1})+(A_{mid}+A_{mid-1}+\ldots+A_L)) \\ +&(mid-L+1)((A_{mid+1})+(A_{mid+1}+A_{mid+2})+\ldots+(A_{mid+1}+A_{mid+2}+\ldots+A_R)) \end{aligned} $$

令 $lsum[p]$ 为该区间所有左端点为 $L$ 的子区间的区间和； $rsum[p]$ 为该区间所有右端点为 $R$ 的子区间的区间和，则综上，有：

$$ fsum[p]=fsum[p1]+fsum[p2]+(R-mid)rson[p1]+(mid-L+1)lson[p2] $$

维护这些信息即可，具体实现细节留给读者编写或参见代码。

代码

#include <cstdio>
#define ri register int
#define rc register char
#define mid ((l+r)>>1)
#define lson p<<1, l, mid
#define rson p<<1|1, mid+1, r
typedef long long int64;
using namespace std;
#ifndef SMALLER
const int SCP=682*999/6;
#else
const int SCP=1000;
#endif
int v[SCP];
int64 fsum[SCP<<2], lsum[SCP<<2], rsum[SCP<<2], sum[SCP<<2], mark[SCP<<2];
int64 fp[SCP];
inline int64 gcd(int64 x, int64 y){ return y?gcd(y, x%y):x; }
inline void getfp(int n){
    fp[1]=1;
    for(ri i=2; i<=n; i++) fp[i]=fp[i-1]+(int64)i*(i+1)/2;
}
inline void mergetree(int p, int p1, int p2, int l, int m, int r){
    fsum[p]=fsum[p1]+fsum[p2]+(r-m)*rsum[p1]+(m-l+1)*lsum[p2];
    lsum[p]=lsum[p1]+lsum[p2]+(r-m)*sum[p1];
    rsum[p]=rsum[p1]+rsum[p2]+(m-l+1)*sum[p2];
    sum[p]=sum[p1]+sum[p2];
}
inline void addtree(int p, int l, int r, int64 x){
    mark[p]+=x;
    fsum[p]+=fp[r-l+1]*x;
    lsum[p]+=(int64)(r-l+1)*(r-l+2)/2*x;
    rsum[p]+=(int64)(r-l+1)*(r-l+2)/2*x;
    sum[p]+=(int64)(r-l+1)*x;
}
inline void up(int p, int l, int r){ mergetree(p, p<<1, p<<1|1, l, mid, r); }
inline void down(int p, int l, int r){ if(mark[p]){ addtree(lson, mark[p]); addtree(rson, mark[p]); mark[p]=0; } }
void update(int p, int l, int r, int a, int b, int64 x){
    if(a<=l and r<=b){
        addtree(p, l, r, x);
        return;
    }
    down(p, l, r);
    if(a<=mid) update(lson, a, b, x);
    if(b>mid) update(rson, a, b, x);
    up(p, l, r);
}
void query(int p, int l, int r, int a, int b){
    if(a<=l and r<=b){
        mergetree(0, 0, p, a, l-1, r);
        return;
    }
    down(p, l, r);
    if(a<=mid) query(lson, a, b);
    if(b>mid) query(rson, a, b);
}
int main(){
    #ifndef ONLINE_JUDGE
    freopen("2221.in", "r", stdin);
    freopen("2221.out", "w", stdout);
    #endif

    ri i, n, m, l, r;
    rc ch;
    int64 x, y, z;

    scanf("%d %d", &n, &m);
    getfp(n-1);
    for(i=1; i<=m; i++){
        do ch=getchar(); while(ch<=32);
        if(ch=='C'){
            scanf("%d %d %lld", &l, &r, &x);
            update(1, 1, n-1, l, r-1, x);
        }
        else{
            scanf("%d %d", &l, &r);
            fsum[0]=lsum[0]=rsum[0]=sum[0]=0;
            query(1, 1, n-1, l, r-1);
            x=fsum[0];
            y=(int64)(r-l)*(r-l+1)/2;
            z=gcd(x, y);
            printf("%lld/%lld\n", x/z, y/z);
        }
    }
    return 0;
}
